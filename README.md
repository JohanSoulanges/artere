# Partie 1

Contexte
Imaginez-vous que vous êtes en milieux professionnel et que vous avez une tâche à réaliser en 2 heures. Le résultat du test nous permettra d'évaluer vos capacités d'analyse et de réalisation d'une tâche.
Dans le test, il s'agit uniquement de développer la partie backend, il n'y a pas besoin de réaliser la partie Front.
vous pouvez utiliser la technologie de votre choix entre Java (avec ou sans Spring ..) ou Javacript ou Python
Nous vous souhaitons bonne chance !
User story 1
En tant que commerçant je souhaite proposer un catalogue de produits sur mon site e-commerce, mon catalogue est constitué de Catégories. Chaque catégorie peut contenir soit des catégories soit des produits, soit les deux

Exercice :
* Créer un modèle pouvant représenter un catalogue de produits
* Je dois pouvoir avoir autant de catégories et de sous-catégories que je souhaite ainsi que de produits
* Une catégorie possède comme propriétés : un nom, une description
* Un produit possède comme propriétés : un nom , un prix , une quantité de stock
* Créer les fonctionnalités permettant d'ajouter modifier et supprimer une catégorie
* Créer les fonctionnalités permettant d'ajouter modifier et supprimer un produit
* Créer les fonctionnalités permettant de lier et délier les catégories et les produi

# Partie 2

En tant que client sur mon site e-commerce ils devront pouvoir : Créer un panier d'achat, ajouter des produits à mon panier , et avoir le total de mon panier


Créer un modèle pouvant représenter un panier
Créer les fonctionnalités nécessaire pour pouvoir ajouter, supprimer, modifier 1 ou plusieurs produits dans un panier ainsi que de visualiser le montant total de mon panier.
